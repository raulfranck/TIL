# arrow function


// Função No java script padrão:
```
function sum(a, b) {
  return (a + b)
}
```


// Arrow function "Completa"
```
const sum2 = (a, b) => {
  return a + b;
}
```

// Quando há apenas uma instrução, podemos reduzir ainda mais o código.
// Removendo a chave do bloco e também a palavra chave return:

```
const sum3 = (a, b) => a + b;
```
